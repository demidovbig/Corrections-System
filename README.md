# Система управления корректировками

API для автоматического повторения ручных корректировок данных, преимущественно строковых.
Полезно для тех алгоритмов, которые регулярно обрабатывают одни и те же / сильно похожие данные.
Система для автоматического повторения ручных корректировок данных с возможностью управления гипотезами и контекстом.

В основе проекта лежала эта идея - https://github.com/den1s0v/vstu_xls/blob/main/materials/corrections.txt.py

##  Требования

- **Node.js** (версия 16 или выше)
- **MySQL** (версия 8.0 или выше)
- **Браузер** (Chrome, Firefox, Edge)

##  Быстрый старт

### 1. Установка зависимостей

```bash
# Установка зависимостей бэкенда
cd ../server
npm install

# Установка зависимостей фронтенда
cd ../client
npm install

# Запуск Бэкенда
cd server
npx nodemon server.js

# Заппуск Фронтенда
cd client
npm start

# Приложение будет доступно по адресу: http://localhost:3000


# Базовые настройки для БД хранятся в корневой папке проекта: Базовые настройки БД.txt
